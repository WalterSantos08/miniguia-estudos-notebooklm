# ☁️ Caderno de Estudos com IA - AWS Cloud Practitioner

---

## 📌 Descrição do Desafio

Este projeto foi desenvolvido como parte de um desafio da DIO (Digital Innovation One), com o objetivo de explorar o uso de ferramentas de Inteligência Artificial, como o NotebookLM, no processo de aprendizado.

A proposta consiste em utilizar IA para estudar um tema específico, registrar os prompts utilizados, analisar as respostas obtidas e organizar todo o conteúdo em um repositório no GitHub.

---

## 📌 Contexto

Este projeto tem como foco o estudo de Computação em Nuvem utilizando a AWS, com o objetivo de preparação para a certificação AWS Cloud Practitioner.

Durante o desenvolvimento, a Inteligência Artificial foi utilizada como apoio para compreensão de conceitos, geração de exemplos e organização do conhecimento.

---

## 🎯 Objetivo

Preparar-se para a certificação AWS Cloud Practitioner por meio de:

- Estudo estruturado dos conceitos de cloud computing  
- Uso de IA para explicações e exemplos práticos  
- Criação de um guia de estudos reutilizável  
- Organização de conteúdos e fontes confiáveis  

---

## ☁️ Conteúdos estudados

- Conceitos de Cloud Computing  
- Modelos de serviço (IaaS, PaaS, SaaS)  
- AWS Well-Architected Framework  
- Segurança na AWS  
- Serviços principais da AWS (EC2, S3, RDS, Lambda)  
- Custos e modelos de cobrança  

---

## 🤖 Uso da Inteligência Artificial

Durante este projeto, ferramentas de IA foram utilizadas para:

- Explicação de conceitos técnicos  
- Geração de exemplos práticos  
- Criação de resumos  
- Apoio na resolução de dúvidas  

O NotebookLM foi utilizado para organizar conteúdos e testar diferentes prompts, permitindo uma aprendizagem mais dinâmica e eficiente.

---

## 📚 Estrutura do Projeto

- `fontes.md` → lista de materiais utilizados  
- `prompts.md` → exemplos de prompts utilizados com IA  
- `mini-guia.md` → resumo final dos estudos
