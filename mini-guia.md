# 📖 Mini Guia - AWS Cloud Practitioner

---

## ☁️ O que é Cloud Computing

Cloud Computing é o fornecimento de recursos de tecnologia pela internet, permitindo acesso sob demanda sem necessidade de infraestrutura física.

---

## 🧱 Modelos de Serviço

* IaaS → infraestrutura
* PaaS → plataforma
* SaaS → software

---

## 🏗️ AWS Well-Architected Framework

Se baseia em 6 pilares:

1. Excelência Operacional
2. Segurança
3. Confiabilidade
4. Eficiência de Performance
5. Otimização de Custos
6. Sustentabilidade

---

## 🔐 Segurança na AWS

* IAM → controle de acesso
* Criptografia
* Responsabilidade compartilhada

---

## ⚙️ Serviços principais

* EC2 → servidores virtuais
* S3 → armazenamento
* RDS → banco de dados
* Lambda → execução sem servidor

---

## 💰 Custos

* Modelo pay-as-you-go
* Paga apenas pelo que usa
* Escalabilidade sob demanda

---

## 🤖 Prompts reutilizáveis

* "Explique [serviço AWS] de forma simples"
* "Compare [serviço A] e [serviço B]"
* "Crie um resumo sobre [tema]"

---

## 🚀 Conclusão

A AWS oferece uma infraestrutura robusta e essencial para profissionais de tecnologia.

O uso de IA no aprendizado acelera a compreensão e melhora a produtividade.
