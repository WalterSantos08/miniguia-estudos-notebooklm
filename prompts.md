# 🤖 Prompts Utilizados

## Prompt 1

Explique o que é Cloud Computing de forma simples

### Resposta

Cloud Computing é o uso de servidores na internet para armazenar, gerenciar e processar dados, sem depender de infraestrutura local.

---

## Prompt 2

Explique a diferença entre IaaS, PaaS e SaaS

### Resposta

* IaaS: infraestrutura como servidores virtuais
* PaaS: plataforma para desenvolvimento
* SaaS: software pronto via internet

---

## Prompt 3

Explique o que é o Amazon EC2

### Resposta

O EC2 é um serviço da AWS que permite criar servidores virtuais na nuvem.

---

## Prompt 4

Explique o que é o Amazon S3

### Resposta

O S3 é um serviço de armazenamento de arquivos altamente escalável e seguro.

---

## Prompt 5

Explique o modelo de responsabilidade compartilhada da AWS

### Resposta

A AWS é responsável pela infraestrutura e o cliente pelos dados e configurações.

---
